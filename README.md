# ThreatLens-AI

ThreatLens-AI is a Flask app to scan suspicious URLs, show a risk score + explanation, store scan history, and display cyber news updates.

## Run locally (Windows PowerShell)

```powershell
venv\Scripts\python.exe run.py
```

Open:
- `http://127.0.0.1:5050/`

## Deploy (Render)

This repo includes `render.yaml` and `wsgi.py`.
